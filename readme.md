#Gobarber web

Implementação em Reactjs do aplicativo web gobarber, feito ao longo do bootcamp da Rocketseat.

##Ferramentas utilizadas

- React
- Unform
- Axios
- Styled-Components
- Typescript
- Yup

##Como usar

Baixe o repositório e dentro da pasta use o comando:

```
npm install

ou

yarn
```
Em seguida execute o comando:
```
npm start

ou

yarn start
```
